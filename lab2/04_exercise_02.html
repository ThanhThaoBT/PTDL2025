Bi 3. Tnh gn ng s pi dng phng php Monte Carlo

V mt ng trn $C$ c bn knh $r$ ni tip mt hnh vung $C$, v t ngu nhin $N$ chm ln hnh vung. T l cc chm nm trong hnh trn trn tng s chm xp x bng din tch ca hnh trn chia cho hnh vung nh sau: $$ \frac{N_C}{N} \approx \frac{\mathbb{S}_C}{\mathbb{S}_N} $$ ni m $N_C$ l cc chm trong hnh trn, $N$ l tng s chm, $\mathbb{S}_C$ v $\mathbb{S}_N$ ln lt l din tch ca hnh trn v hnh vung. $$ \mathbb{S}_C = \pi * r^2 $$ $$ \mathbb{S}_N = (2*r)^2 = 4r^2 $$ Do , cng thc trn tr thnh: $$ \frac{N_C}{N} \approx \frac{\pi * r^2}{4r^2} = \frac{\pi}{4} $$ Suy ra, ta c gi tr s $\pi$ c tnh nh sau: $$ \pi \approx \frac{4N_C}{N} $$

Cc bn hy lp trnh tnh sp x s $\pi$ dng phng php Monte Carlo. Cho bit cc sai s vi $N = 100$, $N = 10000$, v $N = 1000000$. V th minh ha cho chng trnh.

Hng dn: Vit chng trnh pht sinh $N$ im $(x_i, y_i)$ vi $x_i, y_i \in [-1, -1]$. $(x_i, y_i)$ thuc hnh trn tm $(0, 0)$ bn knh 1 ni tip hnh vung c tm ti $(0, 0)$ di cnh 2 khi v ch khi $x_i^2 + y_i^2 <= 1$.

import numpy as np

def calc_pi_monte_carlo(n = 100):
    pi = 0
    
    # gi : s dng np.random.rand(s1, s2, ...) --> sinh ngu nhin cc s trong na on [0.0, 1.0)
    """ CC BN LM BI  Y """    
    # Sinh ngu nhin N im (x, y) trong khong [0, 1) s dng np.random.rand()
    # Sau  chuyn i sang khong [-1, 1] bng cng thc: 2*x - 1
    x = 2 * np.random.rand(n) - 1  # Chuyn [0,1) -> [-1,1)
    y = 2 * np.random.rand(n) - 1  # Chuyn [0,1) -> [-1,1)
    
    # Tnh khong cch t mi im n tm (0,0)
    distances = x**2 + y**2
    
    # m s im nm trong hnh trn (khong cch < 1)
    points_inside = np.sum(distances < 1)
    
    # Tnh xp x s pi
    pi = 4 * points_inside / n
    
    return pi
# calc_pi_monte_carlo

#V  th vi N = 100
def plot_monte_carlo(n=100):
    # Sinh im s dng np.random.rand() theo gi 
    x = 2 * np.random.rand(n) - 1  # Chuyn [0,1) -> [-1,1)
    y = 2 * np.random.rand(n) - 1  # Chuyn [0,1) -> [-1,1)
    
    distances = x**2 + y**2
    
    # Phn loi im trong v ngoi hnh trn
    inside = distances < 1
    outside = ~inside
    
    # Tnh  xp x
    pi_approx = 4 * np.sum(inside) / n
    
    # V  th
    fig, ax = plt.subplots(figsize=(6, 6))
    
    # V hnh trn
    circle = plt.Circle((0, 0), 1, color='g', fill=False, linewidth=2)
    ax.add_patch(circle)
    
    # V im trong hnh trn (mu xanh)
    plt.scatter(x[inside], y[inside], color='blue', s=5, alpha=0.6, label=f'Inside ({np.sum(inside)})')
    
    # V im ngoi hnh trn (mu )
    plt.scatter(x[outside], y[outside], color='red', s=5, alpha=0.6, label=f'Outside ({np.sum(outside)})')
    
    # V im ti tm
    plt.plot(0, 0, 'o', color='black', markersize=5)
    
    plt.xlim(-1, 1)
    plt.ylim(-1, 1)
    plt.title(f'Monte Carlo Simulation (N={n})\n  {pi_approx:.4f}\nSai s: {abs(pi_approx - math.pi):.6f}')
    plt.legend()
    plt.grid(True)
    plt.axis('equal')
    plt.savefig(f'pi_monte_carlo_N_{n}.png')
    plt.show()
    
    return pi_approx
